Installation Instructions:
==================================================
1. Import the "SQL/codeigniter_db.sql" file into the database of your CodeIgniter application.
2. Move all files to the same directory of your CodeIgniter application.
3. Open the "application/config/autoload.php" file ===> Define the library (database and session) and helper (url) to load automatically.
4. Open the URL (https://www.example.com/gallery/) on the browser and test the multiple image upload CRUD functionality in CodeIgniter.
